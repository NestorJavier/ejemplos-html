Documentos y Estructuras HTML

¿Que es HTML?

HTML significa Lenguaje de Marcado para Hipertextos (HyperText Markup Language) y este es usado para estructurar información al nivel más básico en una página web, determina el contenido de esta a través de etiquetados (Markup), los siguients son algunas de las etiquetas, no son todas, pero son de las mas usadas.

<head>, <title>, <body>, <header>, <article>, <section>, <p>, <div>, <span>, <img>

Anatomia de un documento HTML

La anatomia de un documento de html esta compuesta como ya se menciono anteriormente, de etiquetas, estas etiquetas son metadatos de la infomación que se esta desplegando en la pagina web, es decir, estas etiquetas representan información de la información que se esta desplegando en la pagina.

las etiquetas mas importantes dentro de html son las siguientes:

  1. <!DOCTYPE>:

    2. Esta etiqueta indica la versión del documento de html, esta información le indica al navegador como es que debe leer el documento.

  2. <html>:

    3. representa la raiz de un documento HTML. El resto de elementos descienden de este elemento.

  3. <head>:

    4. provee información general acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

  4. <title>:

    5. Dentro de esta etiqueta se indica el nombre del documento el cual va a aparecer en la pestaña del navegador.

  5. <body>:

    6. Representa el contenido de un documento HTML. Sólo puede haber un elemento <body> en un documento.

Encabezados

h1, h2, h3, h4, h5, h6. Estos son los encabezados que se usan en html y estos se usan al igual que en otro tipo de documentos como por ejemplo los periodicos.


Donde h1 seria "MEN WALK ON MOON" y h2 seria "ASTRONAUTS LAND ON PLAIN; COLLECT ROCKS, PLANT FLAG", y asi de forma decendente, estos se usan para dar una idea a grnades rasgos de la información que se visualiza.

Sintaxis:

<h1> "Encabezado" </h1>

<h2> "Encabezado" </h2>

<h3> "Encabezado" </h3>

Divs

Este elemnto se usa para dividir bloques de codigo, esto para aplicarles estilos o realizar operaciones.

Atributos

Estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios. por ejemplo el atributo de un encabezado seria de la siguiente manera

<h3> Encabezado id="ATRIBUTOS" </h3>

donde "id" es un atributo el cual asigna un identificador a ese encabezado en especifico.

Texto y estilo del mismo

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos. Sus etiquetas son: <p> y </p>

Saltos de linea

El elemento HTML line break <br> produce un salto de línea en el texto, y este solo requiere una etiqueta de apertura.

Listas ordenadas y no odrdenadas

Las listas son importantes para ordenar información de una mejor manera, en html existen las listas ordenadas (<ol> </ol>) en las cuales los elementos dentro de la lista estan numerados de menor a mayor y listas no ordenadas (<ul> </ul>) en las cuales los elementos estan resaltados con viñetas, para agregar elementos a estas listas solo se tiene que agregar la etiqueta (<li> </li>) por cada uno de los elementos dentro de la lista. Imagenes

Las imagenes pueden ser incluidas en un archivo de html mediante la etiqueta <img src="" alt=""> donde el atributo src es la fuente de la imagen la cual puede estar contenida dentro de la misma pagina web o en otra pagina, el atributo alt sirve como un texto asociado a la imagen, este texto puede ser una descripción de la misma.

Videos

Esta es la etiqueta que es usada para incluir un video en html <video src=""></video src="">, como se puede observar esta etiqueta es bastante parecida a la de imagenes, ya que contiene tambien el atributo src el que, de igual manera indica la fuente del video.
