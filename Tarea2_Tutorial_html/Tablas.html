Crear Tabla

Para crear una tabla solo hace falta Agregar las etiquetas
<table>

</table>
para este punto solo hemos especificado la creación de la tabla, pero esto no es sufuciente ya que tambien debemos agregar renglones y celdas, si no hacemos esto nuestra tabla no sera visible.

Renglones

En este codigo ya hemos agrgado renglones (<tr> </tr>) y celdas (<td> </td>) y como se puede observar existe un anidamineto en el cual el elemnto mas importante es la etiqueta coorrespondiente a la tabla es la de mayor jerarquia le sigue la etiqueta de renglon y al final el elemento menor perteneciente a la celda.


<table>
    <tr>
        <td>
        </td>
    </tr>

    <tr>
        <td>
        </td>
    </tr>
</table>

Expanción de renglones y columnas

Esto es de utilidad cuando se requiere que una celda o un renglon sean de un tamaño mayor y lo que se tiene que hacer es agregar los atributos rowspan o colspan a la etiqueta de la celda (td).

Cuerpo

El elemento tbody nos sirve para seccionar los renglones de una tabla y de esta manera manteer separada la información cuya tematica es distinta.
